Documentation


Manual for Users:

Steps to Be Followed
1. You can generate and upload XMLs for 2 types of documents called Declaration and Manifest.
2. And there are 2 methods to upload XMLs in this app. One is to fill a single Manifest/Declaration through web forms and the other is to bulk import data through an Excel.
3. You may go to the desired page, i.e form or bulk, and fill the form with available data points or select the files need to be imported.
4. Once the data is filled or files are selected for importing, you can click "Upload" Button to produce the XML that is valid as per given XSD Schema.

Manual for Developers:

Steps to Be Followed
1. Install Docker on your system.
2. Go to the base file directory and run "docker-compose up -d --build"
3. Once the container is up and running, you can access the app at: localhost:5000
